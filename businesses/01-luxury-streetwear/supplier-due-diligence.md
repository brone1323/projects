# Supplier Due Diligence — Luxury Streetwear

## Supplier Approval Criteria

A supplier is only useful for this project if it supports most of the following:

- No inventory purchase.
- No MOQ.
- No monthly minimum.
- No setup fee that blocks launch.
- White-label fulfillment.
- Custom branding options.
- API or technically workable order automation.
- Reliable production and tracking updates.
- Clear return and defect rules.
- Ability to support premium blanks.

## Primary Candidate: Apliiq

### Why It Fits

- Focused on apparel brands rather than basic novelty POD.
- Supports private-label clothing workflows.
- Offers branding options such as labels, embroidery, patches, and packaging workflows.
- Suitable for elevated streetwear and capsule collections.
- Can operate as the fulfillment backend behind a custom storefront.

### Risks To Verify

- Exact API endpoint coverage.
- Whether all branding options are available without minimums.
- Production timelines for branded orders.
- Return rules for made-to-order products.
- Cost impact of labels, embroidery, and packaging.
- International shipping performance.

## Backup Supplier Categories

- Reliable global POD apparel supplier.
- Premium blank supplier with API or order automation.
- Future small-batch manufacturer for best-selling products.

## Required Final Output

Before building production commerce flows, create a supplier matrix including:

- Supplier name.
- Signup link.
- API documentation link.
- No MOQ confirmation.
- Fees.
- Product quality notes.
- Branding features.
- Shipping regions.
- Return policy.
- Agreement risks.
- Approval score.
